# @coreline-engineering-solutions/messaging

Angular messaging library with WebSocket support, Material UI components, and RxJS state handling.

## Single Integration Guide

Use `MESSAGING_INTEGRATION.md` as the one canonical document for:

- install and consumer setup,
- API/WS URL contract,
- `contact_id` requirements,
- CES incident notes and known failure patterns,
- git-install packaging requirements.

## Quick Install

```bash
npm install git+https://github.com/Coreline-Engineering-Solutions/messaging.git#main --legacy-peer-deps
```

## Build (library repo)

```bash
cd messaging-app
npm install
npm run build:lib
```
